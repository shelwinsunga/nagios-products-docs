# Using The Nagiostats Utility

## Using The Nagiostats Utility

A utility called nagiostats is included in the Nagios distribution. It is compiled and installed along with the main Nagios daemon. This documentation explains how the nagiostats utility allows you to obtain various information about a running Nagios process that can be very helpful in tuning performance.
