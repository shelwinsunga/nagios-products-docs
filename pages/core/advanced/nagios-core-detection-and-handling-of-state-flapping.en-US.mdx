# Detection and Handling of State Flapping

## Detection and Handling of State Flapping

Nagios supports optional detection of hosts and services that are "flapping". Flapping occurs when a service or host changes state too frequently, resulting in a storm of problem and recovery notifications. This documentation explains how flapping works.

[Documentation - Detection and Handling of State Flapping](https://assets.nagios.com/downloads/nagioscore/docs/nagioscore/4/en/flapping.html "Detection and Handling of State Flapping")
