# How To Create A Virtual Instance In The Amazon EC2 Cloud Environment

## How To Create A Virtual Instance In The Amazon EC2 Cloud Environment

This document describes how to launch a new pre-installed Nagios® XI™ server in the Amazon EC2 cloud in order to quickly run a trial of Nagios XI without using physical hardware, migrate existing physical installations to a cloud infrastructure, and/or scale an existing XI monitoring environment.
