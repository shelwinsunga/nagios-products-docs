# System Requirements

## Overview

Make sure the server you install Nagios XI on meets the requirements below.

**Supported Distributions:**

Nagios XI is currently supported with the following Linux distributions for 64 bit installations only:

- RHEL 7/8/9
- CentOS 7/Stream 8/ Stream 9
- Oracle Linux 7/8
- Debian 9/10/11
- Ubuntu 16/18/20/22 (LTS)

**Minimum system specs:**

- 1 GHz Processor
- 1 GB RAM
- 8 GB HD

**Recommended system specs:**

- 2+ GHz Processor
- 2 GB RAM
- 40 GB HD
- A redundant disk subsystem
