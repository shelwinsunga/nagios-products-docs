# Installation Options

## Overview

There are two options available for installing Nagios XI:

- [**Use a Nagios XI "pre-made" image (VMware VM, vSphere OVF Template or Microsoft VM).**](https://support.nagios.com/kb/article.php?id=16&show_category=83) This is the fastest way to get up and running with Nagios XI. The VMware machine can be used in enterprise environments where VMware ESX, vSphere or other VMware server products are deployed.

- [**Install Nagios XI manually.**](https://support.nagios.com/kb/article.php?id=17&show_category=83) Use this option if you wish to install Nagios XI on a physical server (e.g. for performance reasons) or a virtual server running a supported Linux distribution.

Choose the option best suited for your needs.Â If in doubt, we recommend using the Nagios XI VMware virtual machine or vSphere / ESX OVF template to get up and running quickly.
